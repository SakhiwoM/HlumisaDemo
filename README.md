# Hlumisa Youth Climate Network Website

Static front-end website concept for **Hlumisa Youth Climate Network** built with HTML, CSS, and JavaScript.

## Structure

- `index.html` - main one-page NGO website
- `assets/css/` - split styling files
- `assets/js/content.js` - editable content data for repeatable cards
- `assets/js/main.js` - interactions, nav, donation modal, forms, scroll reveal
- `assets/images/` - local SVG illustrations and favicon

## Edit Points

- Update core copy and section wording in `index.html`
- Edit reusable cards and placeholder contact details in `assets/js/content.js`
- Replace SVG illustrations in `assets/images/`
- Connect a live payment flow in `assets/js/main.js`

## Deploy

This project is ready for static hosting on GitHub Pages, Netlify, or any standard web server.

## Notes

- Forms are demo-only and do not submit to a backend yet
- Donation flow is a placeholder modal and not a real payment integration
- Content avoids unverified claims and uses safe NGO-style demo wording
